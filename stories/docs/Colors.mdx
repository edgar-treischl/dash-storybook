## Usage in Components

These colors are defined in the `retention.tsx` file in the dash-ui library and used consistently across all components:

```typescript
export const SCHOOL_TYPE_COLORS: Record<SchoolType, string> = {
  'GS': '#264653',   // Dark Blue
  'MS': '#2a9d8f',   // Teal
  'RS': '#e9c46a',   // Gold
  'GY': '#f4a261',   // Orange
  'IGS': '#e76f51',  // Coral
}
```

The color palette is designed to be visually distinct and accessible for data visualization.
