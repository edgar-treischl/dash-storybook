# Welcome to Dash UI Components

This is the component library for the Dash project, featuring reusable React UI components designed for data visualization and analytics.

## Getting Started

### What's included

This library provides a collection of React components built with:
- **React 19+** - Modern React with the latest features
- **TypeScript** - Full type safety and IDE support
- **Vite** - Fast development and building
- **tsup** - ESM and CommonJS distribution formats

### Available Components

- **BarPlot** - Visualize school retention data with interactive bar charts

## Installation

To use these components in your project:

```bash
pnpm add github:edgar-treischl/dash-master/packages/ui
```

## Quick Example

### BarPlot for Data Visualization

```tsx
import { BarPlot } from 'dash-ui'

export function MyComponent() {
  const data = [
    { type: 'GS', count: 120, percent: 75 },
    { type: 'MS', count: 40, percent: 25 },
  ]

  return <BarPlot data={data} year="2024/25" />
}
```

### YearSelect with Icon for Year Filtering

```tsx
import { YearSelect, SelectIcon } from 'dash-ui'
import { useState } from 'react'

export function FilterComponent() {
  const [selectedYear, setSelectedYear] = useState('2024/25')

  return (
    <div style={{ display: 'flex', alignItems: 'center', gap: '8px' }}>
      <SelectIcon />
      <YearSelect 
        value={selectedYear}
        onChange={setSelectedYear}
      />
    </div>
  )
}
```

## Contributing

To add or modify components:

1. Create or edit components in `packages/ui/src/`
2. Export them in `src/index.ts`
3. Add a `.stories.tsx` file alongside the component
4. Test in the playground app (`apps/playground`)
5. Build and distribute: `pnpm build`
